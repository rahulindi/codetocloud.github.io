Project Overview

This project implements a Three-Tier Web Application (ReactJS frontend, NodeJS backend, and MongoDB) deployed on AWS Elastic Kubernetes Service (EKS)​:contentReference[oaicite:0]{index=0}. It is an advanced end-to-end DevSecOps solution incorporating infrastructure as code, continuous integration, delivery, and monitoring. The application is containerized and deployed via Kubernetes, ensuring high availability and scalability.

Key tools explored in the project include Terraform and AWS CLI for infrastructure provisioning, Jenkins and SonarQube for CI/CD automation and code quality, Helm and Prometheus for monitoring, and ArgoCD for GitOps practices​:contentReference[oaicite:1]{index=1}. The pipeline automates IAM user setup, EKS cluster creation, and load balancer configuration, with private ECR repositories for container images and Helm charts to deploy a robust monitoring stack​:contentReference[oaicite:2]{index=2}. This comprehensive setup demonstrates a full DevOps workflow from development to production deployment on Kubernetes.

  • Helm & ArgoCD: GitOps-driven application and monitoring deployment
